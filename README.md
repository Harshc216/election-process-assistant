# Election Process Education Assistant

## Idea
This project helps users understand the election process in a simple and interactive way.

## Features
- Step-by-step election process
- Timeline of election phases
- Interactive chatbot
- Google Maps integration

## Approach
A rule-based chatbot is used to respond to user queries based on keywords.

## How it Works
User enters a question → system matches keywords → returns answer.

## Testing
- Tested chatbot responses manually
- Verified UI in browser
- Checked multiple input cases

## Efficiency
- Lightweight static project
- Fast loading without heavy frameworks

## Assumptions
- Users ask basic election-related queries
